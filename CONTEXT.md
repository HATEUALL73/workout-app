# CONTEXT.md — Приложение «Зал» (трекер тренировок)

## Назначение
Персональное мобильное приложение для отслеживания силовых тренировок.
Только для личного использования владельца. Публикация в Play Market не планируется
(сборка личного apk). Один пользователь, без регистрации и облака.

## Стек
- **Expo (React Native)** + **TypeScript**
- **expo-router** для навигации (file-based routing)
- **expo-sqlite** для локального хранения данных на телефоне
- Без бэкенда, без сети, всё работает офлайн
- Целевая платформа: Android (Samsung Galaxy S24), личный apk через EAS Build

## Дизайн
- Тёмная тема, стиль «качалки»
- Акцентный цвет: оранжево-красный #e8482b
- Фон: #0a0a0c, поверхности: #15151a / #1e1e25
- Текст: #f2f2f5, приглушённый: #8b8b96
- Шрифт системный, крупные тапабельные элементы (управление одной рукой в зале)
- Интерфейс полностью на русском языке

## Экраны (планируемые)
1. **Тренировка** — выбор дня (ПН/СР/ПТ), список упражнений, ввод веса и повторов
   по каждому подходу, отметка «выполнено», кнопка таймера отдыха у каждого упражнения.
2. **Таймер** — обратный отсчёт отдыха между подходами, пресеты 60/90/120/150 сек,
   вибрация и звук по окончании.
3. **Прогресс** — выбор упражнения, график роста рабочего веса по датам,
   статистика (текущий вес, прибавка, максимум, число тренировок), история.

## Модель данных (SQLite)
- `exercises` — справочник упражнений (id, день, название, подходы, повторы,
  время отдыха, примечание по технике, флаг danger для ограничений).
- `logs` — записи выполнения (id, exercise_id, дата, вес, повторы).
  При отметке «выполнено» сохраняется самый тяжёлый подход за тренировку.
- `draft` — текущее (не завершённое) состояние ввода по упражнениям,
  чтобы данные не терялись при выходе.

## Программа тренировок (исходные данные — занести в seed БД)

ВАЖНО: у владельца есть ограничения по здоровью, отражённые в примечаниях:
- Проблемы со спиной: НЕТ становой тяги и приседаний со штангой на спине.
  Жимы/тяги — с опорой на спинку, без осевой нагрузки.
- Повреждён правый мениск: без глубокого сгибания колена под нагрузкой,
  без выпадов, скручиваний и прыжков.

### Понедельник
1. Жим ногами в тренажёре — 3×10-12, отдых 120с
   ⚠ Неполная амплитуда, колено не уходит глубоко за 90° — бережём мениск
2. Жим штанги лёжа — 3×8-10, отдых 150с
   База на грудь. В последнем подходе 1-2 повтора в запасе
3. Тяга верхнего блока к груди — 3×10-12, отдых 90с
   Спина прижата к упору, тянем локтями вниз
4. Жим гантелей сидя (плечи) — 3×10-12, отдых 90с
   ⚠ Только с опорой на спинку — без осевой нагрузки на поясницу
5. Планка — 3×40-60 сек, отдых 60с
   Кор. Нейтральная спина, без прогиба в пояснице

### Среда
1. Сгибание ног в тренажёре лёжа — 3×12, отдых 90с
   Бицепс бедра. Безопасно для спины и колена
2. Жим гантелей на наклонной — 3×10-12, отдых 120с
   Верх груди. Угол скамьи 30-45°
3. Тяга горизонтального блока — 3×10-12, отдых 90с
   ⚠ С упором в грудь, спина зафиксирована
4. Подъём гантелей на бицепс — 3×12, отдых 60с
   Без раскачки корпусом
5. Разгибание на трицепс на блоке — 3×12, отдых 60с
   Локти прижаты к корпусу

### Пятница
1. Разгибание ног в тренажёре — 3×12-15, отдых 90с
   ⚠ ОСТОРОЖНО с мениском: минимальный вес, комфортная амплитуда.
   Боль или щелчки — убрать упражнение
2. Жим штанги узким хватом — 3×8-10, отдых 120с
   Трицепс, вместо брусьев. Хват чуть уже плеч
3. Тяга верхнего блока обратным хватом — 3×10-12, отдых 90с
   Акцент на широчайшие и бицепс
4. Махи гантелями в стороны — 3×12-15, отдых 60с
   Средняя дельта. Лёгкий вес, чистая техника
5. Скручивания на верхнем блоке — 3×15, отдых 60с
   «Молитва». Стоя на коленях (колено на коврике). Можно добавлять вес

## Принципы работы с кодом
- Чистый TypeScript, типизация моделей данных обязательна.
- Компоненты небольшие и переиспользуемые.
- Вся работа с БД вынесена в отдельный модуль (db.ts) с типизированными функциями.
- Комментарии в коде на русском, где поясняют логику.
- Не усложнять: это личный проект, без избыточных абстракций.
